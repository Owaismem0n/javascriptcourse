How to Upload On GitHub

        Step 1: Create New Repository
        Step 2: Copy Remote Link.
        Step 3: Open Terminal on VS Code.
        Step 4: Write These Commands
        
        git init
        git remote add origin (paste link)
        git add .
        git commit -m 'initial commit'
        git pust origin master


        Step 6: Go to Settings of Your GitHub Page and Change Source to 'Master Branch'. 
        Done.
